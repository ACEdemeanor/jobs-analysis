# Job Market and Salary Trends Analysis

## Dataset Source
Dataset obtained from Kaggle:
 [ https://www.kaggle.com/datasets/bismasajjad/global-ai-job-market-and-salary-trends-2025/data]

## Number of Rows
The dataset contains over 8,000 job listings.

## Variables Description
- job_title: Job role/title
- salary: Salary offered
- location: Job location
- skills: Required skills
- posted_date: Job posting date
  

## Tools Used
- Python
- pandas
- Matplotlib
- Seaborn
- GitHub

## Key Findings
- Technical roles offer the highest salaries
- Salary distribution is skewed
- Location impacts job availability and pay
- High demand does not always mean high salary
- Skills significantly influence earnings
